# The Frying Saucer (TFS)

Welcome to the documentation for **The Frying Saucer (TFS)** is a modern burger experience built around choice, customization, and quality.

![TFS logo](assets\images\newTFSlogo.jpg)

## Using TFS application

The Frying Saucer application allows customers to order customizable burgers through the website. Customers can choose their burger patty, burger bun, add or remove toppings, and build a meal that fits their taste. Every burger order includes a side of fries, keeping the ordering experience straightforward while still giving users flexibility in how their burger is made.

## About This Documentation

The documentation provides information regarding the systems, tooling, and processes that power The Frying Saucer’s digital ecosystem. Alongside product information, you’ll find resources for developers and contributors working on TFS applications and services.
