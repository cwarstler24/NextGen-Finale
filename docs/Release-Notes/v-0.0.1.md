# The Frying Saucer Release Notes – Version 0.0.1

## Overview

Version 0.0.1 marks the initial release of The Frying Saucer (TFS), a full-stack burger ordering application that allows customers to browse a menu, customize burgers, add fries, and place orders through a modern web interface.

This release establishes the core frontend and backend architecture, including order processing APIs, a responsive user interface, database persistence, automated testing, and CI/CD pipelines. It also resolves several early defects discovered during development and integration testing.
